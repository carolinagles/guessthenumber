# Guess the Number

**Main Structure:**
- **Main Menu** with 4 options: Single Player, 2 Player, Statistics, Exit
- **Difficulty Submenu** for both game modes: Easy (20 tries), Medium (12 tries), Hard (5 tries)

**Game Rules:**
- Target number is always between **1 and 1000**
- Player gets hints if their guess is too high or too low
- Game ends when player guesses correctly or runs out of attempts

**Data Tracking:**
- After each game, player enters their name
- All results are saved to an **Excel file** for record keeping
- Statistics option lets players view the game history from the Excel data

The game uses core Python concepts like loops, conditionals, random numbers, and file handling to create a complete guessing game experience with progress tracking.
